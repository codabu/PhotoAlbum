Photo Album - Read Me

Author - Corry Burton - 11/26/21

To run the application open PhotoAlbum/bin/Debug/net6.0/PhotoAlbum.exe 

Follow the onscreen prompts. You may select an option between 1 and 100 to view the data of 100 different placeholder photo albums.

To exit the program, type 0 and press the enter key.
